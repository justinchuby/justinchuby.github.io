CMU Course Find

CMU Course Find is a site with information about all CMU courses. It features real-time information and a easy to use search engine. Built with Django in 2015 and completely rewritten using React.js this summer, this web app was born as a 15-112 term project and has been helping students explore and discover cool courses since then.

Check it out: cmucoursefind.xyz

Learn CPR

Did you know that nearly 383,000 out-of-hospital sudden cardiac arrests occur annually, and 88 percent of cardiac arrests occur at home? This means that the life you save with CPR is more likely to be someone you love.[1] CPR stands for Cardiopulmonary resuscitation. It can be performed by anyone, and it's easy to learn!
